# Unit 5: Testing, Handoff & Front-end Basics

---

## 5.1 Usability Testing

### Process
1. Define goals
2. Recruit participants (3-5)
3. Create test tasks
4. Conduct sessions
5. Analyze findings

### Metrics
- Task success rate
- Time on task
- Error rate
- User satisfaction

---

## 5.2 Iteration

Based on testing feedback:
1. Identify issues
2. Prioritize fixes
3. Update designs
4. Re-test if needed

---

## 5.3 Developer Handoff

### Handoff Includes
- Design specs (sizes, colors)
- Assets (images, icons)
- Style guide
- Interaction notes

### Figma Export
- Export assets (PNG, SVG)
- Inspect mode for specs
- Share prototype link

---
