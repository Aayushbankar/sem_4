# Unit 4: Visual Design & Prototyping

---

## 4.1 Typography

### Hierarchy
- H1: Main heading (24-32px)
- H2: Section heading (18-24px)
- Body: Content text (14-16px)

### Font Pairing
- Heading: Sans-serif (bold)
- Body: Sans-serif (regular)

---

## 4.2 Color Theory

| Type      | Use                     |
| --------- | ----------------------- |
| Primary   | Brand color, CTAs       |
| Secondary | Accents                 |
| Neutral   | Text, backgrounds       |
| Semantic  | Success, error, warning |

### Accessibility
- Contrast ratio: 4.5:1 minimum
- Don't rely only on color

---

## 4.3 Prototyping

### Fidelity Levels
| Level  | Interactivity                |
| ------ | ---------------------------- |
| Lo-fi  | Paper, static                |
| Mid-fi | Clickable wireframes         |
| Hi-fi  | Full interaction + animation |

---

## 4.4 Figma Basics

- Frames: Container for designs
- Components: Reusable elements
- Auto Layout: Flexible layouts
- Prototyping: Link screens
