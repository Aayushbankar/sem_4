# Unit 1: Foundations & Design Thinking

## 1.1 UI vs UX

| UI (User Interface) | UX (User Experience) |
| ------------------- | -------------------- |
| Visual design       | Overall experience   |
| Colors, typography  | User journey         |
| Buttons, icons      | Usability            |
| How it looks        | How it feels         |
