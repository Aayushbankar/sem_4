# Unit 3: Information Architecture & Interaction Design

---

## 3.1 Information Architecture (IA)

**IA** = Organizing and structuring content for easy navigation.

### IA Components
- Site maps
- Navigation
- Labeling
- Search

### Card Sorting
- Open: Users create categories
- Closed: Users sort into predefined categories

---

## 3.2 Wireframing

| Fidelity | Detail       | Purpose      |
| -------- | ------------ | ------------ |
| Lo-fi    | Sketches     | Quick ideas  |
| Mid-fi   | Basic layout | Structure    |
| Hi-fi    | Detailed     | Final design |

---

## 3.3 Layouts & Grids

### Grid System
- Columns (12-column common)
- Gutters (spacing between)
- Margins (edge spacing)

### Responsive Breakpoints
| Device  | Width      |
| ------- | ---------- |
| Mobile  | <768px     |
| Tablet  | 768-1024px |
| Desktop | >1024px    |

---

## 3.4 Interaction Patterns

- Forms & validation
- Navigation menus
- Search & filters
- Error handling
- Loading states

---

## 3.5 Nielsen's Usability Heuristics

1. Visibility of system status
2. Match between system and real world
3. User control and freedom
4. Consistency and standards
5. Error prevention
