## 2.1 Research Methods

| Method             | Type         | Description              |
| ------------------ | ------------ | ------------------------ |
| Interviews         | Qualitative  | One-on-one conversations |
| Surveys            | Quantitative | Questionnaires           |
| Observation        | Qualitative  | Watch users in context   |
| Contextual Inquiry | Qualitative  | Interview + observation  |

## 2.2 Personas

**Persona** = Fictional user representing target audience.

### Persona Components
- Name, photo, demographics
- Goals and motivations
- Pain points
- Behaviors and preferences
